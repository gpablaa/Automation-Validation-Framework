# AI-Assisted Coding Workflow

## Tools used

- ChatGPT for project planning, validation rule design, documentation drafts, and debugging support
- GitHub Copilot for Python autocomplete, helper functions, comments, and code refinement

AI tools were used to support prototyping, debugging, documentation, and iteration speed. The final workflow design, validation logic, dashboard structure, and business rules were reviewed, customized, and organized manually.
